# 状态刷新

一个浏览器扩展，用于检测目标网站并在连续停留超过设定时间后自动关闭标签页。支持 Chrome 和 Microsoft Edge。

## 功能

- 根据 URL 或页面标题中的关键词匹配目标网站
- 连续停留超过设定秒数后自动关闭对应标签页
- 切换标签页时计时自动重置
- 通过弹窗界面配置目标关键词和限时时间

## 安装

1. 下载或克隆本仓库到本地
2. 打开浏览器扩展管理页面：
   - Chrome：地址栏输入 `chrome://extensions`
   - Edge：地址栏输入 `edge://extensions`
3. 开启「开发者模式」
4. 点击「加载解压缩的扩展」
5. 选择本项目所在文件夹

## 使用

1. 点击浏览器工具栏中的扩展图标
2. 在弹窗中设置目标关键词（每行一个），例如：
   ```
   douyin.com
   抖音
   ```
3. 设置限时时间（单位：秒），默认 300 秒（5 分钟）
4. 点击「保存配置」

配置保存后，扩展会在后台持续检测当前活动标签页。当页面 URL 或标题匹配任一关键词，且连续停留时间达到设定秒数时，该标签页将被自动关闭。

## 文件结构

```
manifest.json   - 扩展配置文件（Manifest V3）
background.js   - 后台检测与计时逻辑
popup.html      - 配置界面
popup.js        - 配置界面交互逻辑
```
